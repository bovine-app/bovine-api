# Bovine

[![Build Status](https://travis-ci.com/bovine-app/bovine.svg?branch=master)](https://travis-ci.com/bovine-app/bovine)

API server for the Bovine podcatcher-on-Rails.

## Development

### Requirements

- Ruby 2.7
- PostgreSQL 11+

### Usage

```bash
bin/setup
bin/rails s
```

### Testing

```bash
bin/rspec
```
